# 📚 Library Management System (Python)

A simple command-line based Library Management System developed using Python. This project allows users to manage books efficiently by performing operations such as adding, viewing, issuing, and returning books.

---

## 🚀 Features

* ➕ Add new books to the library
* 📖 View all available books
* 📕 Issue books
* 📗 Return issued books
* 📊 Track book availability status

---

## 🛠️ Technologies Used

* Python 3.x
* No external libraries required (pure Python)

## ▶️ How to Run

1. Clone the repository:

```
git clone https://github.com/your-username/library-management-system.git
```

2. Navigate to the project folder:

```
cd library-management-system
```

3. Run the program:

```
python library_management_system.py
```

---

## 🧑‍💻 Usage

After running the program, you will see a menu:

```
1. Add Book
2. View Books
3. Issue Book
4. Return Book
5. Exit
```

* Enter the corresponding number to perform an action.
* Follow on-screen instructions.

---

## 💡 Example

* Add a book with ID, title, and author
* Issue the book using its ID

## ⚠️ Limitations

* No database integration
* Command-line interface only
